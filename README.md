# MoodTracker

MoodTracker is a simple iOS application that lets users track their daily mood with an emoji-based slider and record the date of their entry. This project uses conceps from CPSC 5330 Module 4.

## Features

- **Mood Selection:** Use a slider to select your mood on a scale from 0 to 100.
